# NPI Registration & Setup

Every Oregon therapist needs a **Type 1 (Individual) National Provider Identifier** before applying to any CCO or payer. If you already have an NPI, skip to the verification section below.

## What You Need to Register

- Your Oregon license number (LCSW, LPC, or LMFT)
- Social Security Number
- Date of birth
- Practice address (use your primary business address — mobile practitioners can add locations later)
- Taxonomy code: **104100000X** (Clinical Social Worker) for LCSWs

## How to Apply

1. Go to the [NPPES website](https://nppes.cms.hhs.gov/) and select "Apply for an NPI"
2. Create an Identity & Access (I&A) account if you don't have one
3. Select **Type 1 (Individual)** — this is for individual practitioners, not organizations
4. Complete all required fields including your taxonomy code
5. Submit the application — you'll receive your NPI within 1-2 business days via email

## Choose the Right Taxonomy Code

For LCSWs, the standard taxonomy code is:

| License | Taxonomy Code | Description |
|---------|--------------|-------------|
| LCSW | 104100000X | Clinical Social Worker |
| LPC | 101YM0800X | Mental Health Counselor |
| LMFT | 106H00000X | Marriage & Family Therapist |

Your taxonomy code must match exactly between NPPES, OHA MMIS, and your CAQH profile. Mismatches cause claim rejections.

## Verify Your Existing NPI

If you already have an NPI, verify it's current and correct:

1. Go to the [NPI Registry Lookup](https://npiregistry.cms.hhs.gov/)
2. Search by your name or NPI number
3. Confirm your **name**, **address**, **taxonomy code**, and **license state** are all accurate
4. If anything needs updating, log into NPPES to make changes — updates take 1-2 business days

## Common Issues

- **Name mismatch**: Your NPI name must exactly match your Oregon license. Married name changes require updating both.
- **Wrong taxonomy**: Using a general social work code instead of the clinical code (104100000X) will cause CCO enrollment issues.
- **Expired address**: Update your address whenever your practice location changes.
