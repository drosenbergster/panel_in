# Taxonomy Codes

Your taxonomy code identifies your provider specialty in standardized healthcare systems. It must be consistent across your NPI (NPPES), OHA MMIS enrollment, CAQH profile, and every CCO application. Mismatches are one of the most common causes of claim rejections.

## Codes by License Type

| License | Primary Taxonomy Code | Description |
|---------|----------------------|-------------|
| **LCSW** | 104100000X | Clinical Social Worker |
| **LPC** | 101YM0800X | Mental Health Counselor |
| **LMFT** | 106H00000X | Marriage & Family Therapist |

## Where Your Taxonomy Must Match

Your taxonomy code appears in at least four places. All must match exactly:

1. **NPPES registry** — your NPI record
2. **OHA MMIS enrollment** — Forms OHA 3972 and 3975
3. **CAQH ProView** — your professional IDs section
4. **CCO applications** — each enrollment form you submit

## Common Mistakes

- **Using a general social work code** instead of the clinical social worker code (104100000X) — this creates a mismatch when CCOs verify against your NPI
- **Adding sub-specialty codes** when the primary code is sufficient — only add sub-specialties if specifically required by a CCO
- **Forgetting to update after a license change** — if you upgrade from an associate license to a full license, update your taxonomy across all systems

## How to Verify

1. Look up your NPI at [NPI Registry](https://npiregistry.cms.hhs.gov/)
2. Check the taxonomy code listed
3. Compare against the table above for your license type
4. If incorrect, log into [NPPES](https://nppes.cms.hhs.gov/) and update — changes take 1-2 business days

## Taxonomy Code Lookup

You can browse the full taxonomy list at the [NUCC Health Care Provider Taxonomy](https://taxonomy.nucc.org/) website. Search for "Clinical Social Worker" or your specific license type to confirm the current code.
