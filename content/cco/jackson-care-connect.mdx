# Jackson Care Connect Provider Enrollment

Jackson Care Connect serves Jackson County and uses **CareOregon** for credentialing — the same process as AllCare and Columbia Pacific. If you've already submitted a CareOregon credential packet for AllCare, contact CareOregon to confirm whether it also covers Jackson Care Connect.

## Critical: Non-Contracted Provider Policy

As of **October 1, 2025**, CareOregon no longer allows non-contracted behavioral health providers to bill for outpatient mental health and SUD services. You **must** complete full credentialing before seeing any Jackson Care Connect members.

## Shared CareOregon Credentialing

Jackson Care Connect uses the identical credentialing process as [AllCare](/pathway/allcare). The same documents, same submission address, same timeline.

**If you're also enrolling with AllCare**, ask CareOregon whether one submission covers both:

**Email:** [credentialing@careoregon.org](mailto:credentialing@careoregon.org)

**Phone:** (541) 471-4106 or (888) 460-0185

## Required Documents

| Document | Notes |
|----------|-------|
| OPCA (2025 version) | All pages initialed and dated, attestation signed within 90 days |
| OHP 3108 | Provider Enrollment Request — specify Jackson Care Connect |
| Release of Information | Include "All Contracted Health Plans" |
| Work history | All employment, current positions marked "Current" |
| Professional liability insurance face sheet | Current, showing $1M/$3M coverage |
| Seclusion & Restraint confirmation letter | On practice letterhead, per OAR 410-141-3590 |
| Copy of active Oregon LCSW license | Current, unencumbered |

## Submit to CareOregon

**Email:** [credentialing@careoregon.org](mailto:credentialing@careoregon.org)

## Timeline

Up to **90 days** per ORS 743B.454. If you've already submitted for AllCare, the Jackson Care Connect credentialing may be streamlined — confirm with CareOregon.
